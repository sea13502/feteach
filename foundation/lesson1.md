# 零基础前端入门第一课 入门
我们看到的网页，就是前端开发做的。下面我们就开门见山看我们得第一个网页。
```html
<!DOCTYPE html>
<html lang="en">
<head>
  <title>前端第一课</title>
</head>
<body>
  hello world
</body>
</html>
```
这就是一个最简单的网页。网页的源代码就是由一对对互相嵌套或者并列的标签组成。重点市成对。最重要的是根标签html，里面嵌套了head和body。这构成了最基础的网页结构。
